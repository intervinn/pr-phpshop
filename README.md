## реализация интернет-магазина на php

не имея основного концепта, данный проект представлят из себя платформу с возможностью для пользователей смотреть и делать заказы, а также размещать свои
обьявления. проект реализует:
* минимальную систему аутентификации
* просмотр товаров, добавление товаров в корзину
* размещение, редактирование, снятие с продажи товаров, просмотр полученных заказов
* отзывы к товарам
* контейнеризация: вместо apache используется nginx и alpine образы для легковесности
